# Demo Undangan Ardy & Dila

Template undangan digital statis untuk GitHub Pages, dibuat dengan HTML, CSS, dan JavaScript murni.

## Fitur

- Cover fullscreen dan nama tamu dari URL
- Musik latar serta tombol play/pause
- Countdown dan tombol Google Calendar
- Profil kedua mempelai
- Akad, resepsi, dan Google Maps
- Love story
- Galeri masonry dan lightbox
- Wedding gift dan tombol salin rekening
- Animasi saat scroll, kelopak bunga, dan navigasi bawah
- Responsif untuk ponsel dan desktop

## Cara melihat nama tamu

Tambahkan parameter `to` pada URL:

```text
https://ardyatms.github.io/undangan-dilardy/?to=Bapak%20Lalu%20Pujiatim
```

## Cara memasang di GitHub

1. Cadangkan `index.html` lama.
2. Unggah `index.html` dan folder `assets` dari paket ini ke root repository.
3. Pastikan GitHub Pages menggunakan branch `main` dan folder `/root`.
4. Tunggu 1–3 menit, lalu buka ulang situs dengan hard refresh.

## Bagian yang wajib diganti

- Semua foto demo dari Unsplash
- Lokasi dan tautan Google Maps
- Akun Instagram
- Narasi love story
- Nomor rekening demo
- Foto Open Graph untuk pratinjau WhatsApp
- Musik bila ingin menggunakan lagu pilihan sendiri

## Catatan

Website mengambil foto dan musik secara online. Untuk versi final yang lebih stabil, simpan foto dan musik milik sendiri di dalam folder `assets` lalu ganti URL pada `index.html` dan `assets/css/style.css`.
